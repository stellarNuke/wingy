# Wingy
## A GUI interface for windows package manager Winget



### version 0.0.2_beta

---
## Prequisites
- internet connection 
- winget CLI [winget](https://github.com/microsoft/winget-cli)
---
Features :

- Search Packages
- see installed programs
- uninstall packages
- install packages
- install packages by version
